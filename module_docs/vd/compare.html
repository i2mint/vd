vd.compare

Backend comparison and recommendation tools for vd.

Provides utilities to compare different vector database backends and recommend the best backend for specific use cases.

vd.compare.compare_backends(backends: list[str] | None = None, *, criteria: list[str] | None = None) → dict[str, dict][source]

Compare multiple backends across various criteria.

Parameters:

  • backends (list of str, optional) – List of backend names to compare. If not provided, compares all available backends.

  • criteria (list of str, optional) – Specific criteria to compare: ‘scalability’, ‘persistence’, ‘cloud_support’, ‘local_support’, ‘cost’, ‘performance’. If not provided, shows all criteria.

Returns:

Comparison results with backend names as keys

Return type:

dict

Examples

>>> import vd
>>> comparison = vd.compare_backends(['memory', 'chroma'])  
>>> for backend, info in comparison.items():  
...     print(f"{backend}: {info['scalability']}")

vd.compare.recommend_backend(*, dataset_size: str = 'small', persistence_required: bool = False, cloud_required: bool = False, budget: str = 'free', performance_priority: str = 'balanced') → dict[source]

Recommend backends based on requirements.

Parameters:

  • dataset_size (str, default 'small') – Dataset size: ‘small’ (< 10K docs), ‘medium’ (10K-1M docs), ‘large’ (1M-100M docs), ‘very_large’ (> 100M docs)

  • persistence_required (bool, default False) – Whether data persistence is required

  • cloud_required (bool, default False) – Whether cloud/managed service is required

  • budget (str, default 'free') – Budget constraint: ‘free’, ‘low’, ‘medium’, ‘high’

  • performance_priority (str, default 'balanced') – Performance priority: ‘speed’, ‘scalability’, ‘balanced’

Returns:

Recommendations with ‘primary’, ‘alternatives’, and ‘reasoning’

Return type:

dict

Examples

>>> import vd
>>> rec = vd.recommend_backend(
...     dataset_size='medium',
...     persistence_required=True,
...     budget='free'
... )  
>>> print(rec['primary'])  
'chroma'
>>> print(rec['reasoning'])
