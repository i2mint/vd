vd.backends.memory

In-memory vector database backend.

This backend stores all documents and vectors in memory, making it suitable for prototyping, testing, and small datasets. It provides a simple reference implementation of the backend protocol.

class vd.backends.memory.MemoryBackend(*, embedding_model: Callable[[str], list[float]], **config)[source]

In-memory vector database backend.

Stores all collections and documents in memory. Perfect for testing, prototyping, and small datasets.

Examples

>>> import vd  
>>> client = vd.connect('memory')  
>>> collection = client.create_collection('test')  
>>> collection['doc1'] = "Hello world"  
>>> results = list(collection.search("hello"))

Stores documents in a dictionary and performs brute-force similarity search. Suitable for small datasets and testing.
